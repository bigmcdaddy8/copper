# CL-0010 — TradeRecord Model

## Summary
Define the canonical `TradeRecord` dataclass — the authoritative schema for one trade from entry through close.

## Acceptance Criteria
- `TradeRecord` importable from `captains_log`
- `trade_id` auto-populated with `uuid4().hex`
- `entered_at` auto-populated with UTC ISO datetime string
- All nullable fields accept `None`
- `uv run pytest apps/captains_log -q` passes

## TradeRecord Fields
| Field | Type | Notes |
|---|---|---|
| trade_id | str | Auto-generated uuid4 hex |
| spec_name | str | |
| environment | str | holodeck \| sandbox \| production |
| underlying | str | SPX, XSP, etc. |
| trade_type | str | IRON_CONDOR \| PUT_CREDIT_SPREAD \| CALL_CREDIT_SPREAD |
| expiration | str | ISO date; "" if not placed |
| short_put_strike | float \| None | |
| long_put_strike | float \| None | |
| short_call_strike | float \| None | |
| long_call_strike | float \| None | |
| outcome | str | FILLED \| SKIPPED \| CANCELED \| REJECTED \| ERROR |
| reason | str | Skip/cancel reason |
| errors | list[str] | Default empty list |
| entry_order_id | str | |
| entry_filled_price | float \| None | |
| net_credit | float \| None | |
| tp_order_id | str | |
| tp_limit_price | float \| None | |
| tp_status | str | PLACED \| FAILED \| UNKNOWN \| FILLED |
| tp_fill_price | float \| None | Set by future TP watcher |
| realized_pnl | float \| None | Set by future TP watcher |
| entered_at | str | Auto-generated UTC ISO datetime |
