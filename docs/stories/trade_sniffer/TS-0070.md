## New Client Methods

### `get_option_expirations(symbol: str) -> list[str]`
`GET /markets/options/expirations?symbol={symbol}&includeAllRoots=true`
Returns sorted list of `YYYY-MM-DD` expiration date strings.

### `get_option_chain(symbol: str, expiration: str, greeks: bool = True) -> list[dict]`
`GET /markets/options/chains?symbol={symbol}&expiration={expiration}&greeks={true|false}`
Always returns a list (normalises single-dict responses). Each dict is one option with fields including `symbol`, `strike`, `option_type`, `bid`, `ask`, and a nested `greeks` dict.

### `place_multileg_order(account_id, legs, price, duration="day", tag=None) -> dict`
`POST /accounts/{account_id}/orders` (form-encoded)

Body:
```
class=multileg
type=limit
duration=day
price={price:.2f}
tag={tag}            # omitted if None
leg[0][option_symbol]=...
leg[0][side]=sell_to_open
leg[0][quantity]=1
...
```

`legs` is a list of dicts with keys `option_symbol`, `side`, `quantity`.
Returns the parsed response JSON (contains `order.id` and `order.status`).

### `cancel_order(account_id: str, order_id: str) -> dict`
`DELETE /accounts/{account_id}/orders/{order_id}`
Returns the parsed response JSON.

### `_post(path, data) -> dict` and `_delete(path) -> dict`
Private HTTP helpers. Same throttle/rate-limit pattern as `_get`.

## `options.py` — Pure Helpers

### `build_occ_symbol(underlying, expiry, option_type, strike) -> str`
Builds an OCC symbol string.
`expiry`: `YYYY-MM-DD` | `option_type`: `C` or `P` | `strike`: float
→ e.g. `build_occ_symbol("SPX", "2024-01-19", "P", 4500.0)` → `"SPX240119P04500000"`

### `get_0dte_expiration(expirations: list[str]) -> str | None`
Returns today's date string if present in `expirations`; otherwise returns the nearest future date. Returns `None` if the list is empty.

### `find_delta_strike(chain: list[dict], target_delta: float, option_type: str) -> dict | None`
Finds the option in `chain` of the given `option_type` (`"put"` or `"call"`) whose `abs(greeks.delta)` is closest to `target_delta`. Returns `None` if no options with valid greeks exist.

### `build_sic_legs(chain: list[dict], target_delta: float = 0.20, wing_width: float = 10.0) -> dict | None`
Builds the four legs of a Short Iron Condor:
- Short put: nearest `target_delta` put
- Long put: short_put_strike − `wing_width`
- Short call: nearest `target_delta` call
- Long call: short_call_strike + `wing_width`

Returns a dict:
```python
{
    "short_put": <option dict>,
    "long_put":  <option dict>,   # looked up by strike from chain
    "short_call": <option dict>,
    "long_call":  <option dict>,
}
```
Returns `None` if any leg cannot be found (e.g., greeks unavailable, wing strike not in chain).

### `calc_sic_credit(legs: dict) -> float`
`(short_put.bid + short_call.bid) − (long_put.ask + long_call.ask)`
Rounds to 2 decimal places.

## `demo/scenario1.py`

```python
def run(client: TradierClient, account_id: str) -> dict:
```

1. `expirations = client.get_option_expirations("SPX")`
2. `expiry = get_0dte_expiration(expirations)` — abort with clear message if `None`
3. `chain = client.get_option_chain("SPX", expiry, greeks=True)`
4. `legs = build_sic_legs(chain)` — abort with clear message if `None` (greeks unavailable)
5. `credit = calc_sic_credit(legs)`
6. Build the four-element leg list for `place_multileg_order`
7. `response = client.place_multileg_order(account_id, leg_list, credit)`
8. Return `{"order_id": ..., "credit": credit, "expiry": expiry, "legs": leg_summary}`

## Notes

- `greeks=None` on a chain option means the market is closed or the option has no quote — `build_sic_legs` returns `None` and the scenario command prints a clear "greeks unavailable — run during market hours" message.
- `calc_sic_credit` may return a negative value (net debit) if the market has moved. The scenario command should warn the user but still place the order (the user is testing sandbox behaviour, not trade management).
- Price improvement FAQ answer: after running scenario1 and polling until fill, compare `fill_price` from the event log against the `credit` submitted. Document in `TRADIER_FAQ.md`.
