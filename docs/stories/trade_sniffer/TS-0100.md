## `status` command

```
tradier_sniffer status
```

Opens the local DB and prints two Rich tables:

**Table 1 — Open Trades**

| Trade # | Type | Underlying | Opened At | Orders |
|---|---|---|---|---|
| TRDS_00001_NPUT | NPUT | SPX | 2024-01-15 14:00 | 1 |

**Table 2 — Recent Events (last 20)**

| # | Time | Type | Order ID | Trade ID | Details |
|---|---|---|---|---|---|
| 42 | 14:03:21 | filled | O001 | TRDS_00001_NPUT | {...} |

If the DB does not exist yet (first run before any polling), prints a friendly message and exits 0.

---

## `reset` command

```
tradier_sniffer reset --confirm
```

Requires the `--confirm` flag — exits with error if omitted. Deletes all rows from all five tables (`trades`, `orders`, `trade_order_map`, `event_log`, `poll_state`) and re-seeds `poll_state`. Prints a confirmation message. Does NOT delete the `.db` file itself.
