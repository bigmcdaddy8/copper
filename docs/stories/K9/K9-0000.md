**Phase:** 0 — Foundation
**Status:** Pre-Approved

## Goal

Bootstrap the `apps/K9/` uv workspace member with the minimum skeleton needed for all
subsequent stories: pyproject.toml, package init, CLI entry point (no commands yet), and
a smoke test that verifies `--help` exits 0.

## Tasks

### 1. Create directory tree

```
apps/K9/
├── pyproject.toml
├── src/
│   └── K9/
│       ├── __init__.py
│       ├── __main__.py
│       └── cli.py
└── tests/
    └── test_smoke.py
```

### 2. Write `pyproject.toml`

```toml
[project]
name = "K9"
version = "0.1.0"
requires-python = ">=3.11"
dependencies = [
    "typer>=0.12",
    "rich>=13",
    "python-dotenv>=1.0",
    "httpx>=0.27",
    "bic",
    "holodeck",
]

[project.scripts]
K9 = "K9.cli:app"

[dependency-groups]
dev = ["pytest>=8"]

[build-system]
requires = ["hatchling>=1.24"]
build-backend = "hatchling.build"

[tool.hatch.build.targets.wheel]
packages = ["src/K9"]

[tool.uv.sources]
bic = { workspace = true }
holodeck = { workspace = true }
```

`holodeck` is a runtime dep because `broker_factory.py` imports `HolodeckBroker` when
`environment = "holodeck"`.

### 3. Register in root workspace

In the root `pyproject.toml`, add `"apps/K9"` to `[tool.uv.workspace] members`.

### 4. Write `src/K9/__init__.py`

Empty file (package marker).

### 5. Write `src/K9/__main__.py`

```python
from K9.cli import app

if __name__ == "__main__":
    app()
```

### 6. Write `src/K9/cli.py`

```python
from __future__ import annotations

import typer
from rich.console import Console

app = typer.Typer(
    name="K9",
    help="Automated 0DTE options trade entry system.",
    no_args_is_help=True,
)
console = Console()
```

### 7. Write `tests/test_smoke.py`

```python
"""Smoke tests for K9 CLI scaffold (K9-0000)."""
import subprocess
import sys


def test_help_exits_zero():
    result = subprocess.run(
        [sys.executable, "-m", "K9", "--help"],
        capture_output=True,
        text=True,
    )
    assert result.returncode == 0, f"stderr: {result.stderr}"
    assert "K9" in result.stdout
```

## Notes

- `no_args_is_help=True` prevents a bare `K9` invocation (with no subcommand) from
  silently exiting 0 — it will print help instead.
- `holodeck` is listed as a runtime dependency (not just dev) because
  `broker_factory.py` imports it at module level in later stories.  If this feels
  heavyweight, it can be moved to a lazy import inside `broker_factory.py`; decide in
  K9-0060.
- Run `uv sync --all-groups --all-packages` after adding the workspace member to pick up
  the new package.
